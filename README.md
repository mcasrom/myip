# MyIP

Aplicación web para obtener información y análisis de direcciones IP.

## Requisitos

- Node.js
- npm

## Instalación

git clone https://github.com/mcasrom/myip.git
cd myip
npm install

## Desarrollo

npm run dev

## Compilar

npm run build

## Producción

npm start

## Variables de entorno

cp .env.example .env
